CSS for JavaScript Developers

CSS is easy to get started with, but very difficult to master. There are no error messages, no console.log, no debugger. For so many JavaScript developers, CSS feels like a mysterious black box.

I started writing CSS in 2007, and for a decade, I stumbled my way through it. I knew enough to get by, but I would frequently find myself in bewildering situations. I'd use a snippet I've used 100 times before, but it would randomly do something different. The language felt so inconsistent and unpredictable!

I decided I was sick of feeling baffled by CSS, and started really trying to understand it. Whenever CSS did something I didn't understand, I'd dig into it, settling into the problem like a warm bath. Over the years, so many puzzle pieces snapped into place, and it totally changed the experience of working with CSS!

CSS for JavaScript Developers is a comprehensive course that shares all of the epiphanies I've had about CSS. My goal is to help you build a robust intuition for the language, to change your relationship with CSS!
